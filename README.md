# TextQuest

Простой консольный текстовый квест на C++. Игрок исследует заброшенный дом и прилегающую территорию, взаимодействует с предметами, собирает инвентарь и открывает новые локации, продвигаясь по сюжету.

## Особенности проекта
- Исследование нескольких уникальных локаций  
- Подбор предметов в инвентарь  
- Осмотр предметов и окружения  
- Взаимодействие с объектами, которое открывает новые элементы  
- Скрытые локации, доступные только при выполнении определённых условий  
- Гибкие взаимодействия с объектами, реализованные через лямбда-функции  

## Команды игрока

| Команда | Описание |
|--------|----------|
| `exit` | выйти из игры |
| `help` | показать доступные команды |
| `show inventory` | показать инвентарь |
| `go to <локация>` | перейти в указанную локацию |
| `put in inventory <предмет>` | положить предмет в инвентарь |
| `remove from inventory <предмет>` | убрать предмет |
| `inspect in location <предмет>` | осмотреть предмет в локации |
| `inspect in inventory <предмет>` | осмотреть предмет в инвентаре |
| `inspect location` | осмотреть текущую локацию |
| `show all locations` | показать все локации |
| `show all items in location` | показать предметы в локации |
| `interact in location with <предмет>` | взаимодействовать с предметом в локации |
| `interact in inventory with <предмет>` | взаимодействовать с предметом в инвентаре |

## Примеры игровых механик
- Рюкзак на улице раскрывает скрытые предметы: бутылку воды и карту района  
- Фотография намекает на спрятанный ключ  
- Ржавый ключ открывает закрытый ящик, в котором находится фонарик  
- Фонарик позволяет попасть в новые локации: *Тихая спальня* и *Подвал*  

## Как запустить

**Требования:**
- CMake 3.20+
- Компилятор C++ (GCC/Clang/MSVC)
- Терминал или IDE (CLion/VSCode)

**Запуск:**
- git clone https://github.com/r0masaN/TextQuest.git
- cd TextQuest
- cmake -B cmake-build-debug
- cmake --build cmake-build-debug
- ./cmake-build-debug/TextQuest
